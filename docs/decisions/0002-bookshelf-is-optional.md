# Bookshelf is optional

## Status
Accepted

## Context
The Gem works without a bookshelf. Power users add it later when deep in prep.

## Decision
If `bookshelf.md` exists in profile, inject it into rendered Gem prompt. If not, skip it.

## Consequences
New users aren't blocked by needing to curate a bookshelf upfront. Power users get richer coaching when they add one.
